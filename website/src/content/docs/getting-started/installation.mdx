---
title: Installation
description: Install envaudit globally or use with npx
---

## Requirements

- Node.js >= 18
- No other dependencies

## Install globally

```bash
npm install -g envaudit
```

Then run from any project directory:

```bash
envaudit check
```

## Use with npx (no install)

```bash
npx envaudit check
```

## Verify installation

```bash
envaudit --version
# envaudit 0.1.0
```
