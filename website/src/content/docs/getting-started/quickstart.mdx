## Run it now

No installation needed — just use npx:

```bash
npx @albertoarena/envaudit check
```

This compares `.env` against `.env.example` in your current directory and reports:
- Missing variables (errors)
- Leaked secrets in `.env.example` (errors)
- Undocumented variables (warnings)
- Empty values (warnings)

## CI mode

Exit with code 1 if any errors are found:

```bash
npx @albertoarena/envaudit check --ci
```
